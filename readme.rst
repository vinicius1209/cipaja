# O que é CIPAJÁ? #

CIPAJÁ é um sistema web simplista para eleição de CIPA. Nele os usuários podem inserir funcionários de uma empresa, criar uma CIPA e iniciar um período de votação para a eleição dos "CIPEIROS".

### Tecnologias usadas

- [x] PHP
- [x] Codenighter
- [x] Javascript
- [x] HTML 5
- [x] Bootstrap
- [x] CSS
- [x] MySQL
